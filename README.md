#  Banking Management System (C++ OOPs Project)

A simple, clean, and well-structured Banking System created using **Object-Oriented Programming (OOP) in C++**.  
This project includes **file handling, interest calculation, transaction history, polymorphism, inheritance**, and a **menu-driven interface**.

##  Features

###  Create Accounts  
- Savings Account (3% interest)  
- Current Account (with overdraft limit)

###  Deposit & Withdraw  
- Validations for low balance & overdraft limit

###  Auto Interest  
- Adds 3% interest to all savings accounts

###  Transaction History  
- Every deposit / withdrawal recorded  
- Stored in vector & displayed on request

###  File Handling  
- All accounts saved to `bankdata.txt`  
- Automatically loads data when program starts  
- Automatically saves data when program exits

###  OOP Concepts Used  
- Inheritance  
- Polymorphism (virtual functions)  
- Abstraction (pure virtual base class)  
- Encapsulation  
- Dynamic binding (Account* pointers)
